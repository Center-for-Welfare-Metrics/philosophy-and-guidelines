# WFI Core Analytical Frameworks & Methodological Standards

This repository serves as the definitive source of truth for the methodologies and analytical standards employed by the **Welfare Footprint Institute (WFI)**. 

Our mission requires extreme conceptual clarity and the avoidance of "accidental rightness." To achieve this, we maintain a living set of protocols that govern how we reason, how we use technology, and how we validate our results.


## Usage
These standards are to be applied to all internal research, external publications, and the development of our analytical AI tools.
